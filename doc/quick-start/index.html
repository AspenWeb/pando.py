Step 1: Given POSIX and virtualenv, make a sandbox:

$ mkdir foo
$ cd foo

Step 2: Create and activate a virtualenv in .aspen:

$ virtualenv --prompt="[foo] " .aspen
$ source .aspen/bin/activate

Step 3: Install diesel and aspen from github:

[foo] $ pip install https://github.com/jamwt/diesel/tarball/master
[foo] $ pip install https://github.com/whit537/aspen/tarball/master

Step 4: Create a web page, and start aspen:

[foo] $ echo Greetings, program! > index.html
[foo] $ aspen
Greetings, program! Welcome to port 8080.

Step 5: Check localhost for your new page!
