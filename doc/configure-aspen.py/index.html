Aspen can be configured via configuration scripts in Python syntax. If project_root is defined and it contains a file named configure-aspen.py, then that script will be the last configuration script to run before Aspen starts up. Additional configuration scripts can be specified as a comma-separated list, set in the environment as ASPEN_CONFIGURATION_SCRIPTS or on the command line as --configuration_scripts. There is a website object in each script’s namespace that can be used to configure aspen, including registering hook callbacks.
