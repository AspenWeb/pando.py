The ^L character is an ASCII page break, also called:

  • form feed
  • \f
  • 12
  • 0xC

Aspen uses the ASCII page break to split simplates and hooks.conf into pages. The page break character is a perfect fit semantically, and it is easy to type in Emacs and Vim.

Caret-L

Since tool support is weak outside of Emacs and Vim, you can also use the two characters "^" (caret) and "L" (uppercase ell). Aspen will convert caret-Ls to true page breaks when loading your simplates and hooks.conf. You can mix them interchangeably in the same file.

To include a caret-L in the template portion of an HTML simplate, you can use the escape sequence &#94;L. That’s how the caret-Ls are included on this page, for example.

Tool Support for True Page Breaks

Page breaks are directly supported in Emacs and Vim. For other editors, you may need to change your key bindings to use Ctrl-L for page breaks. Check Wikipedia for a list of editors with dynamic keybinding support.

Ideally, one would be able to configure an editor to use different syntax highlighting for different pages in a file. For Aspen simplates, one would want the first two (optional) pages in a file to be Python, and the third to vary by file extension. I don’t know of any editor that currently supports this.

Web browsers don’t render page breaks, so it is better to use caret-L any time you want to put a simplate code snippet on a website.

Below are some notes for specific editors.

Emacs

To enter a page break in Emacs, type C-q C-l. It renders as ^L.

Emacs actually provides for navigating through a file based on page breaks. I first discovered the page break character in the email package in the standard library, which was written by Barry Warsaw, an Emacs user.

This just in: apparently Emacs can do multiple modes by page without too much trouble.

UltraEdit

There is a “Page Break” option on the “Insert” menu. You should remap Ctrl-L to that. By default, UltraEdit renders page breaks as a nice thin line, though there is an option to render it as an ugly box.

Vim

Page breaks show up in Vim as ^L. You can enter one in insert mode simply by typing Ctrl-L.
